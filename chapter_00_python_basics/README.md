# 第 0 章：Python 科学计算基础

> 本章为 Java/后端开发者量身定制，用类比的方式快速掌握 Python 科学计算生态。

## 本章目标

- 掌握 NumPy 数组操作（机器人数据的基础）
- 学会用 Matplotlib 画时序图和分布图
- 理解 Python 科学计算与 Java 的核心差异

## 文件列表

| 文件 | 内容 | 预计时间 |
|------|------|---------|
| `01_numpy_essentials.py` | NumPy 核心操作，对比 Java 数组 | 45 min |
| `02_numpy_robot_data.py` | 用 NumPy 处理机器人数据场景 | 30 min |
| `03_matplotlib_basics.py` | 数据可视化基础 | 30 min |
| `04_exercises.py` | 综合练习 | 30 min |

## 运行方式

```bash
cd chapter_00_python_basics
python 01_numpy_essentials.py
python 02_numpy_robot_data.py
python 03_matplotlib_basics.py
python 04_exercises.py
```

## 通关标准

- [ ] 能用 NumPy 创建、索引、切片、reshape 数组
- [ ] 能对数组做向量化运算（不写 for 循环）
- [ ] 能用 Matplotlib 画折线图、直方图、子图
- [ ] 完成 04_exercises.py 中的所有练习
