## 1. 向量基础

### 1.1 定义与物理含义

向量是"有方向的量"。在机器人学中，几乎所有物理量都是向量：

| 物理量 | 维度 | 例子 |
|:-------|:----:|:-----|
| 位置 | 3 | 末端执行器在世界坐标系中的位置 |
| 速度 | 3/6 | 线速度 v ∈ ℝ³，或线速度+角速度 ∈ ℝ⁶ |
| 力/力矩 | 3/6 | 末端施加的力和力矩 |
| 关节角度 | n | qpos ∈ ℝⁿ |
| 关节速度 | n | qvel ∈ ℝⁿ |

### 1.2 点积 (Dot Product / Inner Product)

```
a · b = a₁b₁ + a₂b₂ + a₃b₃ = ‖a‖ ‖b‖ cos θ
```

```python
import numpy as np
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
dot = np.dot(a, b)         # 32
dot = a @ b                # 同上，推荐写法
```

**几何含义**: 衡量两个向量的"同向程度"。

| 结果 | 含义 |
|:-----|:-----|
| a · b > 0 | 夹角 < 90°，大致同向 |
| a · b = 0 | 正交（垂直） |
| a · b < 0 | 夹角 > 90°，大致反向 |

**机器人应用**: 判断力的方向与运动方向的关系（做正功 / 做负功 / 不做功）。

### 1.3 叉积 (Cross Product)

仅定义在 ℝ³ 中，结果是一个向量：

```
a × b = ‖a‖ ‖b‖ sin θ · n̂     （n̂ 是右手定则确定的法向量）

展开:
a × b = [ a₂b₃ − a₃b₂ ]
        [ a₃b₁ − a₁b₃ ]
        [ a₁b₂ − a₂b₁ ]
```

```python
c = np.cross(a, b)
```

**机器人应用**:
- Jacobian 中 hinge 关节的贡献: Jₚ 第 i 列 = zᵢ × (p_ee − pᵢ)
- 力矩 = 力臂 × 力: τ = r × F

### 1.4 范数 (Norm)

```
‖x‖₂ = √(x₁² + x₂² + ··· + xₙ²)     L2 范数（欧几里得距离）
‖x‖₁ = |x₁| + |x₂| + ··· + |xₙ|      L1 范数（曼哈顿距离）
‖x‖∞ = max(|x₁|, |x₂|, ..., |xₙ|)    无穷范数
```

```python
np.linalg.norm(x)         # L2（默认）
np.linalg.norm(x, ord=1)  # L1
np.linalg.norm(x, ord=np.inf)  # 无穷范数
```

**机器人应用**: IK 中的误差 ‖x_target − x_current‖ 就是 L2 范数。

### 1.5 单位向量

```
â = a / ‖a‖       长度为 1 的方向向量
```

```python
a_hat = a / np.linalg.norm(a)
```

## 2. 矩阵基础

### 2.1 矩阵是线性变换

矩阵 A ∈ ℝᵐˣⁿ 是一个线性映射: ℝⁿ → ℝᵐ。

```
y = A x

输入 x ∈ ℝⁿ  →  输出 y ∈ ℝᵐ
```

**每一列**是基向量变换后的像：

```
A = [ a₁ | a₂ | ··· | aₙ ]

A eᵢ = aᵢ    （eᵢ 是第 i 个标准基向量）
```

### 2.2 矩阵乘法

```
(AB)ᵢⱼ = ∑ₖ Aᵢₖ Bₖⱼ
```

关键性质：
- **不满足交换律**: AB ≠ BA （一般情况）
- **满足结合律**: (AB)C = A(BC)
- **转置翻转**: (AB)ᵀ = BᵀAᵀ

```python
C = A @ B                  # 推荐
C = np.matmul(A, B)        # 同上
C = np.dot(A, B)           # 也行，但 @ 更清晰
```

### 2.3 转置

```
(Aᵀ)ᵢⱼ = Aⱼᵢ       行列互换
```

```python
A_T = A.T
```

### 2.4 常见特殊矩阵

| 名称 | 性质 | 例子 |
|:-----|:-----|:-----|
| 单位矩阵 I | AI = IA = A | `np.eye(n)` |
| 对角矩阵 | 非对角元素为 0 | `np.diag([1, 2, 3])` |
| 对称矩阵 | Aᵀ = A | 质量矩阵 M、惯性矩阵 |
| 正交矩阵 | AᵀA = I，A⁻¹ = Aᵀ | 旋转矩阵 R |
| 反对称矩阵 | Aᵀ = −A | 叉积矩阵 [ω]× |

### 2.5 NumPy 速查

```python
np.zeros((m, n))           # m×n 零矩阵
np.ones((m, n))            # m×n 全 1 矩阵
np.eye(n)                  # n×n 单位矩阵
np.diag(v)                 # 向量 → 对角矩阵
np.diag(A)                 # 矩阵 → 提取对角元素
A.shape                    # (m, n)
A.T                        # 转置
```

## 3. 线性方程组

### 3.1 Ax = b 的三种情况

```
A ∈ ℝᵐˣⁿ,  x ∈ ℝⁿ,  b ∈ ℝᵐ
```

| 情况 | 条件 | 解 |
|:-----|:-----|:---|
| 唯一解 | m = n，A 满秩 | x = A⁻¹b |
| 无穷多解 | n > m（欠定）或 A 不满秩 | 有自由变量 |
| 无解 | b 不在 A 的列空间中 | 只能求最小误差解 |

```python
x = np.linalg.solve(A, b)     # 仅适用于 A 方阵且满秩
```

### 3.2 最小二乘解

当方程无解（过定系统，m > n）时，找使 ‖Ax − b‖² 最小的 x：

```
法方程:  AᵀA x = Aᵀb
解:      x = (AᵀA)⁻¹ Aᵀb = A⁺ b
```

```python
x, residuals, rank, sv = np.linalg.lstsq(A, b, rcond=None)
```

**机器人应用**: 传感器标定中，测量数据远多于未知参数，用最小二乘拟合。

## 4. 逆矩阵与伪逆

### 4.1 逆矩阵 A⁻¹

仅当 A 是**方阵**且**满秩**时存在：

```
A A⁻¹ = A⁻¹ A = I
```

```python
A_inv = np.linalg.inv(A)
```

性质:
- (AB)⁻¹ = B⁻¹ A⁻¹
- (Aᵀ)⁻¹ = (A⁻¹)ᵀ

### 4.2 Moore-Penrose 伪逆 A⁺

对**任意**矩阵都存在，是逆矩阵的推广：

| A 的形状 | 伪逆公式 | 含义 |
|:---------|:---------|:-----|
| 方阵满秩 | A⁺ = A⁻¹ | 退化为普通逆 |
| 胖矩阵 (n > m) | A⁺ = Aᵀ(AAᵀ)⁻¹ | **最小范数**解: min ‖x‖ s.t. Ax = b |
| 瘦矩阵 (m > n) | A⁺ = (AᵀA)⁻¹Aᵀ | **最小误差**解: min ‖Ax − b‖ |

```python
A_pinv = np.linalg.pinv(A)
```

**机器人应用**: IK 中求关节速度 q̇ = J⁺ ẋ，其中 J⁺ 就是 Jacobian 的伪逆。

> 详见 `notes/jacobian_matrix_deep_dive.md` 第 6 节。

## 5. 行列式与秩

### 5.1 行列式 det(A)

仅对**方阵**定义。

```
2×2:  det(A) = ad − bc     其中 A = ┌ a  b ┐
                                     └ c  d ┘
```

**几何含义**: 线性变换对体积（面积）的缩放因子。

| det(A) | 含义 |
|:-------|:-----|
| det(A) > 0 | 保持方向（不翻转） |
| det(A) < 0 | 翻转方向 |
| det(A) = 0 | 降维（奇异，不可逆） |
| \|det(A)\| = 1 | 保体积（旋转矩阵） |

```python
d = np.linalg.det(A)
```

### 5.2 秩 rank(A)

矩阵中线性无关的行（或列）的最大数目。

```python
r = np.linalg.matrix_rank(A)
```

| 概念 | 含义 |
|:-----|:-----|
| rank(A) = min(m, n) | 满秩 |
| rank(A) < min(m, n) | 秩亏（奇异/接近奇异） |

**机器人应用**: Jacobian 的秩决定了末端能在几个方向上独立运动。rank(J) < 任务维度 → 奇异配置。

## 6. 特征值与特征向量

### 6.1 定义

对方阵 A，若存在非零向量 v 和标量 λ 使得：

```
A v = λ v
```

则 λ 是**特征值**，v 是对应的**特征向量**。

**直觉**: v 是"被 A 变换后方向不变的向量"，只是伸缩了 λ 倍。

### 6.2 计算

```python
eigenvalues, eigenvectors = np.linalg.eig(A)

# 对称矩阵用这个（保证实数特征值，更稳定）
eigenvalues, eigenvectors = np.linalg.eigh(A)
```

### 6.3 关键性质

| 性质 | 公式 |
|:-----|:-----|
| 特征值之和 = 迹 | λ₁ + λ₂ + ··· + λₙ = tr(A) |
| 特征值之积 = 行列式 | λ₁ · λ₂ ··· λₙ = det(A) |
| 对称矩阵的特征值 | 全部是实数 |
| 正交矩阵的特征值 | 绝对值 = 1 |

### 6.4 机器人应用

- **质量矩阵 M 的特征值**: 反映各方向的惯性大小。特征向量是主惯性轴方向
- **刚度矩阵 K 的特征值**: 各方向的刚度大小
- **协方差矩阵的特征值**: PCA，找数据的主方向

## 7. 奇异值分解 (SVD)

### 7.1 定义

**任意**矩阵 A ∈ ℝᵐˣⁿ 都可以分解为：

```
A = U Σ Vᵀ

U ∈ ℝᵐˣᵐ   正交矩阵（左奇异向量）
Σ ∈ ℝᵐˣⁿ   对角矩阵（奇异值 σ₁ ≥ σ₂ ≥ ··· ≥ 0）
V ∈ ℝⁿˣⁿ   正交矩阵（右奇异向量）
```

### 7.2 几何直觉

SVD 把任意线性变换分解为三步：

```
A = U Σ Vᵀ
     ↑  ↑  ↑
     旋转 缩放 旋转

  x  →  Vᵀ旋转  →  Σ缩放  →  U旋转  →  y
```

### 7.3 计算

```python
U, sigma, Vt = np.linalg.svd(A)

# sigma 是一维数组 [σ₁, σ₂, ...]
# 重构: A = U @ np.diag(sigma) @ Vt  （需要处理尺寸）
```

### 7.4 SVD 揭示的信息

| 信息 | 从 SVD 获取 |
|:-----|:-----------|
| 秩 | 非零奇异值的个数 |
| 条件数 | κ = σ_max / σ_min |
| 列空间 | U 的前 r 列 |
| 零空间 | V 的后 n−r 列 |
| 伪逆 | A⁺ = V Σ⁺ Uᵀ |
| 最佳低秩近似 | 截断前 k 个奇异值: A ≈ U_k Σ_k V_k^T |

### 7.5 机器人应用

- **条件数**: κ(J) = σ_max / σ_min 衡量 Jacobian 的奇异程度
- **可操作性**: w = σ₁ · σ₂ ··· σₘ = √det(JJᵀ)
- **伪逆计算**: 通过 SVD 计算更数值稳定
- **数据降维**: 用截断 SVD 压缩高维传感器数据

> 条件数详见 `notes/jacobian_matrix_deep_dive.md` 第 5.3 节。

## 8. 正定矩阵

### 8.1 定义

对称矩阵 A 是**正定**的，当且仅当对所有非零向量 x：

```
xᵀ A x > 0
```

**半正定**: xᵀ A x ≥ 0（允许等号）。

### 8.2 等价条件

以下条件彼此等价（任一成立则全部成立）:

| 条件 | 判定方式 |
|:-----|:---------|
| xᵀAx > 0 对所有 x ≠ 0 | 定义 |
| 所有特征值 > 0 | `np.all(np.linalg.eigvalsh(A) > 0)` |
| 所有顺序主子式 > 0 | 手算用 |
| Cholesky 分解存在 | `np.linalg.cholesky(A)` 不报错 |

```python
# 快速判定
try:
    np.linalg.cholesky(A)
    print("正定")
except np.linalg.LinAlgError:
    print("非正定")
```

### 8.3 机器人应用

- **质量矩阵 M**: 物理上必须正定（动能 = ½ q̇ᵀ M q̇ > 0）
- **QP 优化中的 Hessian**: 正定保证目标函数是严格凸的，有唯一最优解
- **协方差矩阵**: 半正定，特征值反映各方向的方差

## 9. 旋转矩阵与正交矩阵

### 9.1 正交矩阵

矩阵 Q 满足：

```
QᵀQ = QQᵀ = I     即 Q⁻¹ = Qᵀ
```

**性质**:
- 保长: ‖Qx‖ = ‖x‖
- 保角: 向量之间的夹角不变
- det(Q) = ±1

### 9.2 旋转矩阵 R ∈ SO(3)

旋转矩阵是满足**额外约束** det(R) = +1 的正交矩阵：

```
SO(3) = { R ∈ ℝ³ˣ³ | RᵀR = I, det(R) = 1 }
```

性质:
- R⁻¹ = Rᵀ （求逆 = 转置，超快）
- 列向量两两正交且为单位向量
- 组合旋转: R_total = R₂ R₁ （先 R₁ 再 R₂）

### 9.3 基本旋转矩阵

绕 x/y/z 轴旋转角度 θ：

```
Rₓ(θ) = ┌ 1     0       0    ┐
         │ 0   cos θ   −sin θ │
         └ 0   sin θ    cos θ ┘

Rᵧ(θ) = ┌  cos θ   0   sin θ ┐
         │    0     1     0    │
         └ −sin θ   0   cos θ ┘

R_z(θ) = ┌ cos θ  −sin θ   0 ┐
         │ sin θ   cos θ   0 │
         └   0       0     1 ┘
```

### 9.4 轴角 → 旋转矩阵（Rodrigues 公式）

绕单位轴 ω̂ 旋转角度 θ：

```
R = I + sin θ [ω̂]× + (1 − cos θ) [ω̂]×²
```

其中 [ω̂]× 是叉积矩阵（反对称矩阵）：

```
        ┌  0   −ω₃   ω₂ ┐
[ω̂]× = │  ω₃   0   −ω₁ │
        └ −ω₂   ω₁   0  ┘
```

### 9.5 与其他旋转表示的关系

| 表示 | 参数量 | 优点 | 缺点 |
|:-----|:------:|:-----|:-----|
| 旋转矩阵 | 9 (6约束) | 直接变换、复合方便 | 冗余、保正交性难 |
| 欧拉角 | 3 | 直观 | 万向锁 |
| 四元数 | 4 (1约束) | 无万向锁、插值好 | 不直观 |
| 轴角 | 3 | 紧凑 | 奇异（θ=0） |

> 详见 `chapter_03_qpos_deep_dive/02_quaternion_deep_dive.md`。

## 10. 齐次变换矩阵

### 10.1 定义

齐次变换矩阵 T ∈ SE(3) 用一个 4×4 矩阵同时表示旋转和平移：

```
      ┌ R   p ┐       R ∈ SO(3): 旋转矩阵 (3×3)
T  =  │       │       p ∈ ℝ³:    平移向量 (3×1)
      └ 0   1 ┘       底行固定为 [0 0 0 1]
```

```
SE(3) = { T ∈ ℝ⁴ˣ⁴ | R ∈ SO(3), p ∈ ℝ³ }
```

### 10.2 变换一个点

```
┌ p' ┐     ┌ R   t ┐ ┌ p ┐     ┌ Rp + t ┐
│    │  =  │       │ │   │  =  │        │
└  1 ┘     └ 0   1 ┘ └ 1 ┘     └   1    ┘
```

即 p' = Rp + t。

### 10.3 链式变换

多个坐标系之间的变换可以链式相乘：

```
T_world_ee = T_world_base · T_base_link1 · T_link1_link2 · ··· · T_linkn_ee
```

这就是**正运动学**的本质——每个关节贡献一个变换矩阵，依次相乘得到末端位姿。

### 10.4 逆变换

```
T⁻¹ = ┌ Rᵀ   −Rᵀp ┐
       └  0      1  ┘
```

不需要通用矩阵求逆，利用 R⁻¹ = Rᵀ 直接构造，速度快且数值稳定。

### 10.5 NumPy 操作

```python
T = np.eye(4)
T[:3, :3] = R          # 填入旋转
T[:3, 3] = p           # 填入平移

R = T[:3, :3]          # 提取旋转
p = T[:3, 3]           # 提取平移

T_inv = np.eye(4)      # 求逆
T_inv[:3, :3] = R.T
T_inv[:3, 3] = -R.T @ p
```

### 10.6 在 MuJoCo 中

MuJoCo 不直接存储齐次矩阵，而是分开存：

```python
# body 的位姿
pos  = data.xpos[body_id]              # 平移 (3,)
rot  = data.xmat[body_id].reshape(3,3) # 旋转矩阵 (3,3)
quat = data.xquat[body_id]             # 四元数 (4,)

# 自己组装齐次矩阵
T = np.eye(4)
T[:3, :3] = rot
T[:3, 3] = pos
```

## 11. 向量空间：列空间、零空间、值域

### 11.1 四个基本子空间

对矩阵 A ∈ ℝᵐˣⁿ，存在四个关联的子空间：

```
                    A: ℝⁿ → ℝᵐ

输入空间 ℝⁿ                      输出空间 ℝᵐ
┌──────────────────┐          ┌──────────────────┐
│  行空间            │    A     │  列空间（值域）     │
│  Row(A) = Col(Aᵀ) │ ──────→ │  Col(A) = Range(A) │
│  dim = r           │          │  dim = r           │
│                    │          │                    │
│  零空间            │          │  左零空间           │
│  Null(A)          │          │  Null(Aᵀ)          │
│  dim = n − r      │          │  dim = m − r       │
└──────────────────┘          └──────────────────┘

r = rank(A)
```

### 11.2 列空间 Col(A)

A 的列的所有线性组合构成的空间 = A 能输出的所有向量。

```
Col(A) = { Ax | x ∈ ℝⁿ }
```

Ax = b 有解 ⟺ b ∈ Col(A)。

### 11.3 零空间 Null(A)

被 A 映射到零的所有输入：

```
Null(A) = { x ∈ ℝⁿ | Ax = 0 }
```

```python
from scipy.linalg import null_space
N = null_space(A)    # 列是零空间的基
```

### 11.4 在 Jacobian 中的意义

| 子空间 | 对应 Jacobian 的含义 |
|:-------|:-------------------|
| Col(J) | 末端能实现的速度方向 |
| Null(J) | 关节动但末端不动的运动（"内部运动"） |
| rank(J) = m | 满秩：末端各方向都能动 |
| rank(J) < m | 奇异：某些方向动不了 |

冗余机器人（n > m）的零空间利用：

```
q̇ = J⁺ ẋ  +  (I − J⁺J) q̇₀
     ─────     ──────────────
     完成任务    零空间中优化
               （避障/远离限位/最小能耗）
```

## 12. 速查表

### 12.1 公式速查

| 概念 | 公式 |
|:-----|:-----|
| 点积 | a · b = ∑ aᵢbᵢ = ‖a‖‖b‖cos θ |
| 叉积（结果向量） | a × b，方向由右手定则确定 |
| 矩阵逆 | AA⁻¹ = I，(AB)⁻¹ = B⁻¹A⁻¹ |
| 伪逆（胖矩阵） | A⁺ = Aᵀ(AAᵀ)⁻¹ → 最小范数解 |
| 伪逆（瘦矩阵） | A⁺ = (AᵀA)⁻¹Aᵀ → 最小误差解 |
| SVD | A = UΣVᵀ |
| 条件数 | κ = σ_max / σ_min |
| 特征值 | Av = λv |
| 正定 | xᵀAx > 0 ∀x ≠ 0 |
| 旋转矩阵 | Rᵀ = R⁻¹，det(R) = 1 |
| 齐次变换逆 | T⁻¹ 的旋转部分 = Rᵀ，平移部分 = −Rᵀp |
