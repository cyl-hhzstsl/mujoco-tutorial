# 第 8 章 · 02 — 机器人控制基础

> **目标**: 理解 PD/PID 控制器如何驱动机器人，以及控制参数如何影响数据特征。

## 核心知识点

1. PD 控制: τ = Kp·e + Kd·ė
2. 重力补偿: 抵消重力对控制的影响
3. 轨迹跟踪: 跟踪正弦波参考轨迹
4. P / PD / PID 控制器性能对比

---

## 1. 控制律

\[
\tau = K_p \cdot (q_{des} - q) + K_d \cdot (\dot{q}_{des} - \dot{q}) + K_i \cdot \int (q_{des} - q) \, dt + \tau_{gravity}
\]

| 项 | 作用 |
| :- | :--- |
| Kp·e | 比例项：误差越大，力矩越大 |
| Kd·ė | 微分项：阻尼振荡，抑制超调 |
| Ki·∫e | 积分项：消除稳态误差 |
| τ_gravity | 重力补偿：`data.qfrc_bias[:nv]` |

### 增益配置

```python
ControllerGains.p_only(kp=[200, 150])
ControllerGains.pd(kp=[200, 150], kd=[20, 15])
ControllerGains.pid(kp=[200, 150], kd=[20, 15], ki=[50, 40])
```

---

## 2. 重力补偿

没有重力补偿 → 机械臂因重力下垂 → 数据中出现**稳态偏移**。

```python
if self.gravity_comp:
    tau += data.qfrc_bias[:nv]  # MuJoCo 内置重力/科氏力计算
```

## 3. P / PD / PID 数据特征对比

| 控制器 | 稳态误差 | 振荡 | 数据特征 |
| :----- | :------: | :--: | :------- |
| P-only | **大** | 多 | 数据有系统性偏移 |
| PD | 可能有 | **少** | 数据更平滑 |
| PID | **无** | 可能有积分饱和 | 最接近参考轨迹 |

### 典型指标

| 指标 | 含义 |
| :--- | :--- |
| RMSE | 均方根误差（越小越好） |
| 最大误差 | 最差情况偏离程度 |
| 平均绝对误差 | 整体跟踪精度 |

### 积分饱和防护

```python
self.integral_error = np.clip(
    self.integral_error, -integral_limit, integral_limit
)
```

## 5. 数据工程视角

| 观察 | 对数据的影响 |
| :--- | :---------- |
| P 控制的稳态误差 | 数据有系统性偏移，校验时会触发偏离告警 |
| PD 的平滑响应 | 数据更适合学习，帧间跳变更少 |
| PID 的积分饱和 | 控制信号可能出现异常峰值 |
| 重力补偿 | 有/无会导致完全不同的数据分布 |
| 控制频率 (dt) | 步长越小 → 数据越密 → 存储成本增加 |

│                                                              │

│                                                              │
│  关键参数:                                                    │
│    Kp 越大 → 响应越快但可能振荡                               │
│    Kd 越大 → 越平滑但响应变慢                                 │
│    重力补偿 → 消除重力导致的数据偏移                           │
└──────────────────────────────────────────────────────────────┘
