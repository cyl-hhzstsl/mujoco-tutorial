# 第 8 章 · 03 — Sim-to-Real 概念

> **目标**: 理解仿真与真实之间的差距（sim-to-real gap），掌握域随机化技术及其对数据分布的影响。

## 核心知识点

1. Sim-to-Real Gap 的来源
2. 域随机化 (Domain Randomization)
3. DomainRandomizer 类
4. 数据增强技术
5. 参数敏感性分析

---

## 1. Sim-to-Real Gap

| 仿真 | 真实世界 |
| :--- | :------- |
| 完美的刚体物理 | 柔性、间隙、磨损 |
| 精确参数 (质量=1.0kg) | 参数不确定 (0.9~1.1kg) |
| 无传感器噪声 | 噪声和延迟 |
| 完美的执行器 | 摩擦、死区 |
| 确定性仿真 | 环境随机变化 |

→ **仿真数据分布 ≠ 真实数据分布** → 模型性能下降

---

## 2. 域随机化 (Domain Randomization)

核心思想：在仿真中**随机改变物理参数** → 数据分布更宽 → 覆盖真实世界的变化范围。

### DomainRandomizer 用法

```python
randomizer = DomainRandomizer(base_xml, seed=42)
randomizer \
    .add_mass_range("g1", low=0.5, high=2.0) \
    .add_friction_range("g1", low=0.3, high=2.0) \
    .add_damping_range("j1", low=0.1, high=2.0) \
    .add_gravity_range(low=-10.5, high=-9.0)

model, params = randomizer.sample()  # 返回随机化后的模型 + 参数记录
```

### 支持的参数

| 参数 | API | 影响 |
| :--- | :-- | :--- |
| 质量 | `body_mass` | 惯量、加速度、重力影响 |
| 摩擦 | `geom_friction` | 接触行为 |
| 阻尼 | `dof_damping` | 运动衰减 |
| 步长 | `opt.timestep` | 积分精度 |
| 重力 | `opt.gravity` | 静力学偏移 |

### 参数敏感性

质量 > 摩擦 > 阻尼（对数据分布的影响程度排序）

---

## 3. 数据增强 (Data Augmentation)

`RobotDataAugmentor` 对已有轨迹进行增强：

| 方法 | 模拟场景 | 代码 |
| :--- | :------- | :--- |
| `add_noise(std)` | 传感器噪声 | 高斯噪声叠加 |
| `time_stretch(factor)` | 不同执行速度 | 时间轴插值 |
| `joint_offset(max)` | 校准误差 | 恒定偏移 |
| `smooth(window)` | 低通滤波 | 滑动平均 |
| `random_crop(ratio)` | 数据多样性 | 随机裁剪片段 |
