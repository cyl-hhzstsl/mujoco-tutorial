# 第 6 章：数据质量与校验

> 数据质量是数据平台的核心价值。垃圾进，垃圾出 —— 再好的模型也无法从低质量数据中学到有用的策略。

## 本章目标

1. 建立一套**完整的数据校验流水线**，在数据入库前拦截问题
2. 掌握**异常检测**方法，识别物理不合理的轨迹片段
3. 学会计算和分析**数据集统计特征**，量化数据分布
4. 使用**可视化工具**快速评估数据质量

## 文件结构

| 文件 | 内容 |
|------|------|
| `01_data_validator.py` | 数据校验器：NaN/Inf、四元数归一化、关节限位、跳变检测、轨迹长度、死关节、时间戳单调性、动作范围、图像维度 |
| `02_data_statistics.py` | 数据统计：关节级/轨迹级/数据集级统计、相关性分析、频谱分析、分布偏移检测 |
| `03_visualization_suite.py` | 可视化工具箱：时间序列、异常标注、分布直方图、相关性热图、速度热图、箱线图、3D 轨迹、质量仪表盘 |
| `04_anomaly_detection.py` | 异常检测：z-score、IQR、滑动窗口、Isolation Forest、物理一致性、时序连续性 |
| `05_exercises.py` | 练习：自定义校验规则、数据清洗流水线、质量评分、批量质量报告 |

## 核心理念

```
原始数据  ──→  校验 (Validator)  ──→  统计 (Statistics)  ──→  可视化 (Visualization)
                   │                       │                        │
                   ▼                       ▼                        ▼
              拦截坏数据              量化数据分布              直观展示问题
                   │                       │                        │
                   └───────────────────────┼────────────────────────┘
                                           ▼
                                    质量报告 / 质量分数
```

## 运行方式

```bash
# 依次运行每个脚本（每个脚本都会自动生成测试数据）
python 01_data_validator.py
python 02_data_statistics.py
python 03_visualization_suite.py
python 04_anomaly_detection.py
python 05_exercises.py
```

## 企业级数据质量的关键维度

| 维度 | 说明 | 本章覆盖 |
|------|------|----------|
| **完整性** | 数据是否缺失字段或帧 | ✅ |
| **一致性** | 数据格式、范围是否统一 | ✅ |
| **准确性** | 数据是否符合物理规律 | ✅ |
| **时效性** | 数据是否及时更新 | — |
| **唯一性** | 是否有重复数据 | ✅ |
| **有效性** | 数据是否在合理范围内 | ✅ |
