# 第 6 章 · 03 — 数据质量可视化工具箱

> **目标**: 提供一套完整的可视化工具，帮助快速识别数据质量问题，一图胜千言。

## 核心知识点

1. 关节角度时间序列 + 限位标注带
2. 异常点高亮标注
3. 关节角度分布直方图
4. 关节相关性热图
5. 帧间速度热图
6. 跨 episode 箱线图
7. 基座 3D 轨迹图
8. 质量评分仪表盘

---

## 1. QualityVisualizer 使用方式

```python
viz = QualityVisualizer(
    output_dir="quality_report",
    joint_names=["J1", "J2", ...],
    hz=50.0,
)

# 一键生成所有 8 张图表
viz.plot_all(episodes, joint_limits_lower, joint_limits_upper)

# 或单独调用
viz.plot_joint_timeseries(qpos, ...)
viz.plot_anomaly_overlay(qpos, anomaly_frames)
viz.plot_correlation_heatmap(qpos)
```

## 2. 八种可视化详解

### 2.1 关节角度时间序列

每个关节一行，纵轴 = 关节值，横轴 = 时间。

- **绿色半透明带**: 关节允许范围
- **红色虚线**: 限位边界
- **红色填充**: 超限区域

→ 一眼看出哪些帧违反了关节限位。

### 2.2 异常高亮叠加

在时间序列上用**红色散点**标注异常帧。

```python
anomaly_frames = {
    0: [50, 51, 52],     # 关节 0 的异常帧
    3: [100, 150],       # 关节 3 的异常帧
}
```

### 2.3 分布直方图

每个关节一个子图，区分范围内（蓝色）和超限（红色）的数据。

### 2.4 相关性热图

关节间 Pearson 相关系数的矩阵热图：
- 红色 = 正相关（同步）
- 蓝色 = 负相关（反向）
- 白色 = 无相关

### 2.5 帧间速度热图

横轴 = 时间步，纵轴 = 关节。颜色 = 帧间速度的对数值。

- **突然的亮条** = 不连续性（跳变、丢帧）
- **均匀暗色** = 平稳运动

### 2.6 跨 episode 箱线图

对每个关节，绘制不同 episode 的值域箱线图。可以发现某些 episode 的分布异常偏离。

### 2.7 3D 轨迹图

基座或末端执行器的 3D 位置轨迹，颜色渐变表示时间进展，标注起点（绿色）和终点（红色）。

### 2.8 质量评分仪表盘

综合仪表盘包含 5 个子图：
- 总体评分（大字显示）
- 各维度质量条形图
- episode 评分分布直方图
- 关节健康度条形图
- 校验通过/失败矩阵
