## 练习 1: 自定义校验规则引擎

用 `ValidationRule` + `RuleEngine` 实现灵活的规则定义和执行。

### 规则定义方式

```python
engine.add_lambda_rule(
    name="no_nan",
    check_fn=lambda d: not np.isnan(d["observations"]["qpos"]).any(),
    severity=Severity.ERROR,
    description="qpos 中发现 NaN 值",
)
```

### 7 条示例规则

| 规则 | 级别 | 检查内容 |
| :--- | :--- | :------- |
| no_nan | ERROR | qpos 无 NaN |
| no_inf | ERROR | qpos 无 Inf |
| min_length | ERROR | 帧数 ≥ 50 |
| joint_range | WARNING | \|qpos\| < π×1.1 |
| no_dead_joints | WARNING | 所有关节 std > 1e-6 |
| monotonic_timestamps | ERROR | 时间戳递增 |
| continuity | WARNING | 帧间跳变 < 2.0 |

## 练习 2: 数据清洗流水线

### DataCleaner 修复策略

| 异常类型 | 修复方法 | 可恢复？ |
| :------- | :------- | :------: |
| NaN / Inf | 线性插值填充 | ✓ |
| 突刺 (spike) | 用相邻帧均值替代 | ✓ |
| 帧间跳变 | 移动平均平滑 | 部分 |
| 死关节 | 仅标记，不修复 | ✗ |

### 核心原则

1. **不修改原始数据**（返回副本）
2. **记录所有修改操作**（可追溯）
3. **保守修复**：只修复有把握的问题
4. **不可恢复的问题只标记不修复**

## 练习 3: 质量评分指标 (0-100)

### 五个维度，每维 20 分

| 维度 | 满分 | 核心判据 |
| :--- | :--: | :------- |
| 完整性 | 20 | 无 NaN/Inf，数据非空 |
| 范围合规 | 20 | 关节值在限位内 |
| 平滑度 | 20 | 帧间跳变小 |
| 关节活性 | 20 | 所有关节有有效运动 |
| 时序一致 | 20 | 时间戳递增，速度合理 |

### 等级映射

| 总分 | 等级 |
| ---: | :--- |
| ≥ 90 | A |
| ≥ 80 | B |
| ≥ 70 | C |
| ≥ 60 | D |
| < 60 | F |
